## 서비스 소개
- **Riot** : 라이엇 API를 활용한 채팅 및 알림 제공 서비스
- 게임 시작 전, 볼일을 보다가 시작을 놓친 적이 있나요?

## Commit Message Convention

> 커밋태그: 내용 #닉네임

ex. `feat : login 파일 추가 #aeeazip'

| command | mean |
| --- | --- |
| feat | 새로운 기능 추가 |
| fix | 버그 수정 |
| docs | 문서 수정 |
| style | 코드 포맷 변경 |
| test | 테스트 코드 추가 |
| refactoring | 코드 리팩토링 |
| chore | 빌드, 패키지 매니저 수정 |
